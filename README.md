# JWT Authentication & Protected Routes

A full-stack authentication system built using React, Node.js, Express, MongoDB, JWT, and bcrypt.

## 🚀 Features

* User Signup
* User Login
* Password Hashing using bcrypt
* JWT Token Authentication
* Protected Backend Routes
* Protected Frontend Routes
* React Router Navigation
* Tailwind CSS Modern UI
* Logout Functionality

---

## 🛠️ Tech Stack

### Frontend

* React.js
* React Router DOM
* Axios
* Tailwind CSS
* Vite

### Backend

* Node.js
* Express.js
* MongoDB
* JWT (jsonwebtoken)
* bcryptjs
* dotenv
* cors

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/suba-2406/jwt-authentication-protected-routes.git
```

---

## Backend Setup

Install dependencies:

```bash
npm install
```

Run backend server:

```bash
npm run dev
```

---

## Frontend Setup

Move into frontend folder:

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

Run frontend:

```bash
npm run dev
```

---

## 🔐 Security Notes

* Passwords are securely hashed using bcrypt.
* JWT tokens are used for authentication and protected routes.
* Tokens are stored in localStorage for demo purposes.
* In production, httpOnly cookies are recommended for better security.
